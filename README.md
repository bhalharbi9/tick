# توقعات دوري الأبطال - نسخة GitHub Pages + Supabase

نسخة نهائية ثابتة، متوافقة مع الجوال، ومناسبة للرفع على GitHub Pages.

## الملفات
- `index.html` صفحة التوقعات
- `leaderboard.html` صفحة الترتيب المباشر
- `admin.html` لوحة المشرف
- `schema.sql` إنشاء الجداول والسياسات وواجهة الترتيب
- `seed.sql` تعبئة أولية للمباريات
- `assets/js/config.js` إعدادات المشروع

## الإعداد السريع
1. أنشئ مشروعًا في Supabase.
2. افتح **SQL Editor** ونفّذ `schema.sql`.
3. نفّذ `seed.sql`.
4. افتح `assets/js/config.js` وضع:
   - `SUPABASE_URL`
   - `SUPABASE_ANON_KEY`
   - `ADMIN_PASS`
5. ارفع الملفات إلى GitHub.
6. فعّل GitHub Pages.

## ملاحظات مهمة
- هذه النسخة **لا تعتمد على PHP**.
- إدخال النتائج يتم يدويًا من `admin.html`.
- الحماية في صفحة المشرف **واجهة فقط** لأن المشروع ثابت. للحماية القوية استخدم Edge Functions لاحقًا.
- تم حذف التشكيلة نهائيًا حسب طلبك.
- نظام النقاط يعتمد على قرب النتيجة:
  - 10 نقاط للنتيجة الدقيقة
  - 7 نقاط إذا كان فارق الأهداف والاتجاه صحيحين
  - 5 نقاط إذا كان الاتجاه صحيحًا
  - +3 إذا تم توقع أهداف أحد الفريقين بدقة
  - +2 إذا كانت المسافة 1
  - +1 إذا كانت المسافة 2

## الصفحات
### index.html
- الاسم فقط
- حفظ واسترجاع التوقعات
- إغلاق تلقائي قبل المباراة

### leaderboard.html
- Auto Refresh
- Top 5
- الجوائز:
  - الأول 30 ريال
  - الثاني 20 ريال
  - الثالث 10 ريال
- تصدير CSV

### admin.html
- تعديل نتائج الذهاب
- إدخال نتائج الإياب
- تعديل وقت المباراة
- تغيير دقائق الإغلاق
